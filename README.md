# WEB103_Hien_Vo_Project_1_EuroGetaway

Simple Express + EJS listicle app featuring nine European destinations.

## Run locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Routes

- `/` - destination grid homepage with client-side search
- `/destinations/:slug` - destination detail pages
- unknown routes return a custom 404 page
